<iframe>

tag is used to embed another document within the current HTML document.

<link>

tag is used to link external resources like CSS files or icons.

<meta>

tag is used to provide metadata about the HTML document.

<script>

tag is used to define client-side JavaScript code.

<canvas>

tag is used to draw graphics on the web page.

<noscript>

tag is used to define alternative content for users who have disabled scripts in their browser.

<style>

tag is used to define CSS styles for the HTML document.

<template>

tag is used to define a template for HTML content.

<track>

tag is used to specify audio or video tracks.

<code>

tag is used to define a section of code within the HTML document. console.log("Hello, World!");

<kbd>

tag is used to define keyboard input.
